# 打包机 HMI 协议对接快速指南

## 1. 当前有效链路

- 左链路（端口 A）：`USART3 / 9600 / 8N1`
- 协议：固定 `20` 字节帧 + `CRC16-Modbus`（低字节在前）
- 功能码：仅保留 `0x40~0x4C`
- HMI 口径：只发送/接收 `20` 字节主协议，不解析 DGUS

- 右链路（端口 B）：`USART1 / 9600 / 8N1`
- 协议：`DGUS 5A A5` 变量协议
- 用途：参数调试 + 系统信息读取 + DGUS 日志显示
- HMI 口径：只发送/接收 DGUS，不解析 `20` 字节主协议

## 2. 打包机主协议（20B）

帧格式：

| 字节 | 含义 |
|---|---|
| 0 | 地址（常用 `0xFA`） |
| 1 | 功能码（`0x40~0x4C`） |
| 2~17 | 数据域（16字节） |
| 18~19 | CRC16-Modbus（L/H） |

## 3. DGUS 调参与日志

- 参数区：`0x2000~`（每参数占2个word）
- 系统信息：`0x1000~0x1003`
- 日志区：`0x3000~0x301F`
- 控制按键：`0x4000~0x4002`
- `0x200E` 对应 `0x17 / speed_self_check_dual_hz`
- 加热参数示例：`0x2080=0x50/加热占空比周期(ms)`，`0x2082=0x51/加热占空比(‰, 0~200=0~20.0%)`
- 扒口角度：`0x207C=0x4E/扒口角度(0.1°)`
- 拉断长度：`0x207E=0x4F/拉断长度(0.001mm, 默认 40.000mm)`

> **已移除的 DGUS 参数：** 以下机械固定参数已改为固件编译期宏，不再通过 DGUS 读写：
>
> - `0x2060~0x2066`（出袋机械尺寸，ID 0x40~0x43）
> - `0x2070~0x2076`（压杆机械尺寸，ID 0x48~0x4B）
> - `0x2084`（ID 0x52，历史开关标志位域）
> - `0x20C0~0x20C4`（ID 0x70~0x72，监控/喂狗/日志周期）

日志显示口径：

- 固件将日志封装为 DGUS `0x82` 写变量帧写入 `0x3000~0x301F`
- HMI 仅解析 DGUS 帧中的日志变量更新，不依赖原始 ASCII 文本行
- Android 自动日志落盘写入应用私有目录；手动导出时由用户选择目标位置，HMI 将 `protocol_logs.txt`、原始滚动日志 `jsonl`、栈统计 `jsonl` 一并打包为 `zip`
- 栈水位相关新口径：
  - `STACK_LEVEL=<min>`：用于 HMI 提取单值样本
  - `STACK_SNAPSHOT_BEGIN`
  - `STACK_TASK NAME=<TaskName> TOTAL=<words> FREE=<words>`
  - `STACK_SNAPSHOT_END`
- 当前任务快照口径：
  - 任务集合固定为 `ProtoTask / StateMachineTask / MotorTask / AdcTask / CommTask / MonitorTask / HeaterTask`
  - `TOTAL` / `FREE` 单位均为 FreeRTOS 栈字数（word，STM32F1 当前为 4 字节/word）
  - HMI 仅在收齐一整份快照后刷新任务表格与结构化栈统计 `jsonl`

系统信息读取口径：

- 固件不再周期性主动写 `0x1000~0x1003`
- HMI 需要按需发送 `0x83` 读请求获取系统信息

## 4. 门禁规则

DGUS 参数读写仅在以下条件满足时允许：

- 状态机 `IDLE`
- 运行许可为 `STOP`
- 启动序列已完成

否则固件只消费帧，不执行业务写入。
